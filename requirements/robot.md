### RF - Requisitos Funcionais
- [ ] deve ser possível determinar para onde o robo está apontando
- [ ] deve ser possível determinar onde o robo está posicionado
- [ ] deve ser possível inicializar o robo informando em qual canteiro ele está 
- [ ] deve ser possível girar o robo para norte, sul, leste, oeste
- [ ] deve ser possível movimentar o robo 1 posição pela horta
- [ ] deve ser possível enviar comando para irrigar canteiro

### RNF - Requisitos Não Funcionais
- [ ] 

### RN - Regras de negócio
- [ ] não deve ser possível se mover para outra direção a não ser a qual o robo está apontado
- [ ] não deve ser possível mover o robo para um canteiro inexistente
- [ ] só deve ser possível irrigar canteiro no qual o robo está situado
- [ ] o robo pode irrigar o canteiro mais de uma vez
